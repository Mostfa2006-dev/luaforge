Connect LuaForge to Cursor IDE, VS Code, or any MCP-compatible AI tool. Generate your personal mcp.json config and paste it into your editor.

Your MCP Config
Log in to generate your config...
How to use with Cursor
1. Copy the config above
2. In Cursor: Settings → Features → MCP Servers → Add
3. Paste the config
4. Cursor will connect to LuaForge as an MCP server
5. You can now ask Cursor to generate Roblox scripts and push them to Studio
How to use with VS Code (Copilot)
1. Install the GitHub Copilot extension
2. Open .vscode/mcp.json in your workspace root
3. Paste the config above
4. Copilot Agent Mode will now have access to LuaForge tools
